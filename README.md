Calculator App

A simple, clean, and responsive calculator built with HTML, CSS, and JavaScript.
This project performs basic arithmetic operations and supports both button clicks and keyboard input.

✨ Features
➕ Addition, subtraction, multiplication, and division
⌨️ Full keyboard support (Enter, Backspace, Escape, numbers, operators)
🧼 Clear/reset functionality
🎯 Real-time input display
⚡ Fast and lightweight (no libraries needed)
📱 Responsive UI design
🛠️ Built With
HTML5
CSS3
JavaScript (Vanilla JS)
🚀 How to Run the Project
Clone the repository:
git clone https://github.com/covenantudofe-creator/calculator-app.git
Open the folder:
cd calculator-app
Open index.html in your browser:
Double click it
OR
Use Live Server (VS Code extension)
📁 Project Structure
calculator-app/
│
├── index.html      # Main structure
├── style.css       # Styling
└── script.js       # Logic
🎮 Keyboard Controls
Numbers: 0–9
Operators: + - * /
Enter → Calculate result
Backspace → Delete last input
Escape → Clear screen
